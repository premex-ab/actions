# actions
Internal actions for the premex organization

## Available Actions

### update-gradle-wrapper
Updates Gradle wrappers in your repository to the latest version (or a specified version).

```yaml
- uses: premex-ab/actions/update-gradle-wrapper@v1
  with:
    gradle-version: 'latest'  # optional, defaults to 'latest'
```

See [update-gradle-wrapper/README.md](update-gradle-wrapper/README.md) for detailed documentation.
